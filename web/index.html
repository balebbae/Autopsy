What this does

Brings up postgres, the FastAPI service on :4000, and the dashboard on :3000, and installs the opencode plugin into the current project. Prompts once for an optional Gemini API key (press Enter to skip). Add -s -- --plugin-only to skip the stack, or -s -- --no-prompt to skip the key prompt.
